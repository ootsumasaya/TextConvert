# TextConvert
クリップボードにコピーしたテキストを.(ピリオド)でのみ改行したテキストに変換します。

論文などをコピーしたときの無駄な改行を消したいときに利用してください。

# 使い方
０．bin/Release/TextConvert.exe をダウンロードして実行。

１．変換したいテキストをクリップボードにコピーしペーストボタンを押す、または右欄に入力する。

２．コピーを押す

これで変換後のテキストがクリップボードにコピーされます。
